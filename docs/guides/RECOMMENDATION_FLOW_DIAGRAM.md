## Summary

This recommendation system uses a **simple scoring algorithm** to provide personalized song suggestions:

1. **Collect** user preferences (liked, played, recent)
2. **Extract** patterns (artists, keywords)
3. **Score** candidate songs (artist: +10, keyword: +3)
4. **Return** top 12 unique recommendations
5. **Display** with modern UI and like functionality

**Result**: Users discover new songs matching their taste without complex AI!
